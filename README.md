------------------------------------------------------------
1. Overview
------------------------------------------------------------
This e-portfolio contains all required components for the
Research Methods and Professional Practice module. It presents
my academic work, reflections, research preparation, and
statistics tasks in a structured HTML format. Pages are linked
through a consistent navigation menu for ease of review.

The portfolio demonstrates my engagement with research ethics,
professional practice, literature review skills, collaborative
discussion, statistical reasoning, and reflective learning.

------------------------------------------------------------
2. How to View the Portfolio
------------------------------------------------------------
The e-portfolio is delivered as a set of HTML pages.

To view:
1. Extract the zip folder (if applicable).
2. Open the file named: index.html
3. Use the navigation bar at the top of each page to access
   Discussions, Research & Proposal, Statistics, Reflections,
   Activities, and About sections.
